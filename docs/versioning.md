# Versioning

Coworking Service Desk uses SemVer.

## Stages

- `v0.1.0-alpha`: initial platform foundation.
- `v0.2.0-alpha`: domain and admin expansion.
- `v0.3.0-beta`: SLA, reports, TestSprite hardening.
- `v1.0.0`: production stable.

## Conventional Commits

Allowed types:

- `feat`
- `fix`
- `docs`
- `refactor`
- `test`
- `ci`
- `chore`
- `perf`
- `security`
- `infra`

## Changelog

Follow Keep a Changelog with:

- Added
- Changed
- Deprecated
- Removed
- Fixed
- Security

## Releases

Releases are created from `main` through `.github/workflows/release.yml`.
